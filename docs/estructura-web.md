# Estructura web

Este documento describe las secciones principales que formarán la primera versión de ARENAS MOTOCICLETAS.

## 1. Hero

- Introducción clara de la marca.
- Llamada a la acción para ver catálogo y pedir cotización.

## 2. Buscador de moto ideal

- Permite filtrar por línea, tipo de uso o cilindrada.
- Será útil para mostrar modelos recomendados rápidamente.

## 3. Catálogo destacado

- Presenta los modelos clave de la marca.
- Extrae inventario del JSON en `data/catalogo.json`.

## 4. Modelo destacado

- Enfoque en una moto principal para ventas directas.
- Incluye CTA hacia WhatsApp.

## 5. Líneas de motos

- Muestra familias: Pulsar, Dominar, Boxer, CT.
- Ayuda a estructurar el catálogo por segmento.

## 6. Por qué elegir ARENAS

- Argumentos de confianza, respaldo local y servicio.
- Refuerza la propuesta premium y deportiva.

## 7. Tu moto incluye

- Beneficios incluidos en la compra.
- Mensaje de valor sobre entrega y servicio.

## 8. Financiamiento

- Datos básicos de cuotas y opciones.
- Configurable para agregar promociones reales.

## 9. Servicio técnico

- Taller propio y mantenimiento en Cusco.
- Ventajas del servicio postventa.

## 10. Promociones

- Espacio para ofertas temporales.
- Ideal para campañas comerciales.

## 11. Testimonios

- Validación social de clientes.
- Mejora la confianza en la compra.

## 12. Tiendas

- Ubicaciones físicas y horarios.
- Contacto directo con sedes.

## 13. Cotización

- Enlaces directos a WhatsApp.
- Formas rápidas de consulta.

## 14. Footer

- Navegación secundaria.
- Enlaces legales.
