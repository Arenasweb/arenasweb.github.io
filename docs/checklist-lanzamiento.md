# Checklist de lanzamiento

## Antes de publicar

- [ ] Revisar contenido y copys de cada sección.
- [ ] Confirmar que los enlaces de WhatsApp usan el número correcto.
- [ ] Actualizar datos de contacto en `data/configuracion.json`.
- [ ] Verificar que `index.html` carga `style.css` y `script.js`.
- [ ] Añadir imágenes reales en `assets/`.
- [ ] Revisar enlaces legales y textos temporales.

## Validación técnica

- [ ] Probar en navegadores modernos.
- [ ] Confirmar que no hay errores de link roto.
- [ ] Validar que el sitio se sirve correctamente desde GitHub Pages.
- [ ] Revisar `robots.txt` y `sitemap.xml`.

## Publicación en GitHub Pages

- [ ] Activar GitHub Pages en la configuración del repositorio.
- [ ] Seleccionar `main` o rama predeterminada como fuente.
- [ ] Confirmar que la URL pública es `https://arenasweb.github.io/`.
- [ ] Revisar la versión publicada y corregir cualquier error visual.
