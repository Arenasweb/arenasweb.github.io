# ARENAS MOTOCICLETAS

Primera versión gratuita de la página oficial de ARENAS MOTOCICLETAS para GitHub Pages.

## Descripción

Sitio estático basado en HTML, CSS y JavaScript puros. Preparado para mostrar el catálogo, opciones de financiamiento, servicio técnico y contacto por WhatsApp.

## Tecnología usada

- HTML5
- CSS3
- JavaScript puro
- JSON para datos de catálogo

## Estructura del proyecto

- `index.html` - página principal
- `style.css` - estilos base
- `script.js` - inicialización JS y placeholders
- `data/` - datos del catálogo y configuración
- `legales/` - páginas legales temporales
- `docs/` - documentación interna del proyecto
- `assets/` - carpetas para imágenes y recursos futuros
- `robots.txt` y `sitemap.xml` - archivos de soporte para SEO

## Cómo abrir localmente

1. Abrir la carpeta del proyecto en Visual Studio Code.
2. Usar la extensión Live Server o un servidor estático local.
3. Abrir `index.html` en el navegador.
